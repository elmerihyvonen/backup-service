# https://www.bluematador.com/learn/aws-cli-cheatsheet#S3

Tiedoston lähetys:

aws s3 --endpoint-url https://eu2.contabostorage.com/ cp backup/mirthdb.2022-07-11-130114.dump.sql s3://backup-docker-mirth

Tiedoston poistaminen:

aws s3 --endpoint-url https://eu2.contabostorage.com/ rm backup/mirthdb.2022-07-11-130114.dump.sql s3://backup-docker-mirth

Bucketin sisällön listaus:

aws s3 --endpoint-url https://eu2.contabostorage.com/ ls s3://backup-docker-mirth

sync:

aws s3 --endpoint-url https://eu2.contabostorage.com/ sync backup/ s3://backup-docker-mirth


ls only filenames:
aws s3 --endpoint-url https://eu2.contabostorage.com/ ls s3://backup-docker-mirth | awk '{print $4}'

S3 lifecycle policies
https://docs.contabo.com/docs/products/Object-Storage/HowTo/lifecycle-policy

aws --endpoint-url https://eu2.contabostorage.com s3api put-bucket-lifecycle-configuration --bucket backup-docker-mirth --lifecycle-configuration file://lifecycle-policy.json
